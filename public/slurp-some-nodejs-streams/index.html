slurp-some Nodejs streams

Posted by: Seth Lakowske

slurp-some is a Nodejs convenience module I published that allows a user to read a stream up to some predefined amount and error if the stream contains more data. It is especially useful when receiving http data from the outside world which can get wild. A malicious person on the internet might try to fill your memory with bullshit data unless you specify a limit to the amount of data youâ€™re willing to receive, which is what slurp-some does for you.
